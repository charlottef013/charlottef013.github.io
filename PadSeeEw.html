Pad See Ew

Although pad see ew is a popular Thai dish, it is believed that it came from Chinese immigrants in thailand. It was created in the 1930s. Pad see ew is a sir fry combined with noodles, chicken, egg, broccoli, sweet soy sauce, and a choice of meat. Pad See ew is my go to at any Thai restaurant. It is always interesting to see how each restaurant changes their recipe. If you had not had it yet, please go try it because it is a must.
